# Egetprojekt Elias Karlström

#### För att köra spelet, så skriver man "python3 guessinggame.py" om man klonar mitt repo. Man kan även ladda ner spelet bara spelet guessingame.py och köra det via terminalen om man går till den mapp där spelet ligger, även där via "python3 guessinggame.py"

Jag har valt att göra ett spel där du gissar på en siffra och spelet svarar sedan om du är för lågt eller för högt.
Mitt spel började med bara en svårighetsgrad (0-100) men jag har sedan lagt till så att det nu finns 3 svårighetsgrader.
Jag ändrade sedan så att spelet håller koll på hur många försök som användaren har gjort för att sedan ge användaren ett betyg. Om man får betyger S så får man ett easter egg.
Mitt spel och kod är relativt simpel, detta var med mening så att jag med min begränsade kunskap ändå kunde ha någon förståelse över vad de olika sakerna gör vad som ändras när jag väl andrar något.
